Deferred Rendering Pipeline — 4 Pass 流程

滑鼠點擊任一 Pass 可查看詳細說明

Pass 1: Geometry

輸出：G-Buffer

  • gbuffer0: Normal(oct) + Roughness + Metallic
  • gbuffer1: Albedo.rgb + AO
  • main: Depth (D32)

Shader: world_gbuffer.kong (MRT)

→
Pass 2: Lighting

輸入：gbuffer0 + gbuffer1 + main(depth)

輸出：buf (lit color)

Shader: deferred_light.kong

單一 directional light + constant ambient

→
Pass 3: Post-FX

鏈：SSAO → Bloom → TAA → last

Phase 3 才實作

→
Pass 4: 2D Overlay

輸入：last

Sprite batch (forward, unchanged)

組件：render2d_bridge.c

G-Buffer 格式

TargetFormat內容
mainD32Depth buffer（所有 Pass 共享）
gbuffer0RGBA64Normal.xy (octahedron), Roughness, Metallic
gbuffer1RGBA64Albedo.rgb, AO
bufRGBA64Intermediate / lighting output
lastRGBA64Final composited output
