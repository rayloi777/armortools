Engine 3D 系統重構 — 總覽

Follows Flecs System + Minic-native design

Layer 架構

Minic Scripts (.minic)
遊戲邏輯腳本 — 創建實體、添加組件、查詢系統
↓ calls C API registered via minic_register()
Core API (core/)
entity_api · component_api · light_api · camera_api · culling_api
↓ operates on
Flecs ECS (ecs/)
系統：culling · shadow · gbuffer · lighting · postfx
↓ syncs to
Iron Engine (base/)
GPU render targets · render_path · mesh_object · camera_object

Flecs ECS World — 所有 3D 組件和系統

Components（ECS 組件）

// Transform comp_3d_position — float x,y,z comp_3d_rotation — float x,y,z,w (quat) comp_3d_scale — float x,y,z // Mesh comp_3d_mesh_renderer — mesh_path, material_path RenderObject3D — iron_mesh_object ptr comp_lod — distances[4], mesh_lod0-3 // Camera comp_3d_camera — fov, near, far, perspective comp_3d_ortho — left,right,bottom,top // Lights comp_directional_light — dir, color, strength, shadow comp_point_light — pos, color, range, shadow comp_spot_light — pos, dir, color, range, cone, shadow // Rendering (tags) Tag: CompRenderable — 可被渲染的實體 Tag: CompShadowCaster — 投射陰影 Tag: CompShadowReceiver — 接收陰影 Tag: CompVisible — 當前幀可見（culling 結果）

ECS Systems（執行順序）

System Order (tick order): 1. sys_culling_frustum // GPU frustum culling → writes CompVisible tag 2. sys_culling_lod // LOD selection → updates comp_lod.current_lod 3. sys_shadow_directional // CSM shadow maps → renders to shadow_0/1/2 4. sys_shadow_point // Point cubemap shadows → renders to shadow_cube_[N] 5. sys_shadow_spot // Spot shadow maps → renders to shadow_spot_[N] 6. sys_gbuffer // Geometry pass → outputs gbuffer0 + gbuffer1 + main(depth) 7. sys_lighting // Deferred lighting → reads gbuffer + all shadow maps → outputs buf 8. sys_postfx // Post-processing → SSAO → Bloom → TAA → last 9. sys_output // 2D overlay + present

Minic API — 方便使用的 C API

// === 燈光 API === entity_create_light_directional(dir_x,dir_y,dir_z, color, strength) → id entity_create_light_point(pos_x,y,z, color, range, strength) → id entity_create_light_spot(pos_x,y,z, dir_x,y,z, color, range, cone) → id light_set_shadow(light_id, cast_shadows, bias) light_set_enabled(light_id, bool) // === 攝影機 API === entity_create_camera3d(fov, near, far) → id camera3d_set_ortho(camera_id, left,right,bottom,top) camera3d_set_active(camera_id) // 成為主攝影機 // === 渲染 API === culling_set_enabled(bool) // 開關 frustum culling culling_set_shadow_only(bool) // 只做 shadow pass culling render_set_pipeline("deferred") // 切換 render pipeline // === 查詢 === query_foreach("CompRenderable", "CompVisible", callback) query_foreach("comp_directional_light", callback)
