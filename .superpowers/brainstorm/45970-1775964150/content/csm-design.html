Cascade Split（對數分割）

split_i = near * (far / near)^(i / N) Cascade 0: 0.1m → ~4.6m (2048×2048, 近景細節) Cascade 1: 4.6m → ~21.5m (1024×1024, 中景) Cascade 2: 21.5m → 100m (512×512, 遠景)

Light Space 計算

light_view: look-at(light_dir, scene_center, up) light_proj: ortho(-extent, extent, -extent, extent, cascade_near, cascade_far) SMVP_i: light_proj × light_view × model_matrix

Cascade 選擇邏輯

idx = (view_depth < split_0) ? 0 : (view_depth < split_1) ? 1 : 2; // 邊界過渡混合 blend = smoothstep(split_i - transition, split_i, depth);

4-tap PCF Shader

float shadow_pcf(sampler2D sm, vec4 sc, float bias) { vec2 ts = 1.0 / shadow_map_size; float s = 0.0; s += texture(sm, sc.xy + vec2(-ts.x,-ts.y)).r; s += texture(sm, sc.xy + vec2( ts.x,-ts.y)).r; s += texture(sm, sc.xy + vec2(-ts.x, ts.y)).r; s += texture(sm, sc.xy + vec2( ts.x, ts.y)).r; s /= 4.0; return s < sc.z - bias ? 0.0 : 1.0; }

Lighting Pass 中的陰影計算

// 選擇 cascade float ci = get_cascade_index(view_depth); vec4 sp = get_shadow_pos(ci); // SMVP[ci] × world_pos // 4-tap PCF float shad = shadow_pcf(shadow_map[ci], sp, light.bias); // NdotL × shadow × light_color color += NdotL * shad * light.color * albedo;
