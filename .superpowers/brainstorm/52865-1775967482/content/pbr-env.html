PBR 環境光 — IBL / Environment Lighting

G-buffer 只儲存 material，環境光需要另外處理

問題

G-buffer 儲存的是 material properties： - Normal, Albedo, Roughness, Metallic, AO - 沒有 lighting 資訊 Deferred Lighting Pass 只能計算 direct lighting（燈光直射） 缺失的環境光： 1. 天空顏色（sky ambient） 2. 間接光（interreflection, bounce light） 3. 反射（environment reflections）

環境光三層次

層次內容計算方式成本
Ambient簡單常數或天空顏色albedo × ambient × ao極低
SH Irradiance球諧函數預計算的間接光albedo × SH_probe(pos, N)低
IBL ReflectionEnvironment map 反射env_map.sample(R, roughness)中

完整 IBL 計算公式

// Diffuse (SH Irradiance) float3 diffuse = albedo * SampleSH(normal) * ao; // Specular IBL (Environment Map) float3 R = reflect(-V, N); float mip = roughness * (num_mips - 1); float3 specular = environment_map.SampleLevel(R, mip); // PBR specular BRDF float3 F0 = lerp(float3(0.04), albedo, metallic); float3 F = FresnelSchlickRoughness(NdotV, F0, roughness); float3 kD = (1 - F) * (1 - metallic); float3 ambient = kD * diffuse + specular * F; // 最終 color = ambient + direct_lighting;

Environment Map 生成

Editor 或 runtime 生成預過濾的 environment map： Cubemap → Convolve with GGX → Mipmap levels 每個 mip level 對應不同的 roughness： Mip 0: roughness = 0.0 (完美鏡面) Mip 1: roughness = 0.1 Mip N: roughness = 1.0 (完全漫反射) 的好處：一次採樣即可得到正確的 specular IBL （不需要複雜的 importance sampling）

天空盒（Skybox）

天空盒也是一個環境光源。需要渲染到 cubemap 或在 shader 中計算。

Phase 1（推薦）：固定天空顏色 float3 sky_color = float3(0.5, 0.7, 1.0); // 藍天 Phase 2：Procedural skybox（在 shader 計算） - Rayleigh / Mie scattering 模擬大氣 - 太陽方向 + 強度 Phase 3：Cubemap skybox（烘焙或 runtime 生成） - 從場景某位置捕獲 cubemap - 預計算 irradiance map + prefiltered env map

球諧函數（SH）Irradiance

SH 是一種壓縮環境光照的方法，用 9 個 float 係數（3 channels × 3 bands）表示 irradiance。

// 9 coefficients per channel (27 floats total) struct SHIrradiance { float3 L00, L1m1, L10, L11, L2m2, L2m1, L20, L21, L22; }; // 採樣 float3 SampleSH(SHIrradiance sh, float3 N) { return sh.L00 + sh.L1m1 * (N.y) + sh.L10 * (N.z) + sh.L11 * (N.x) + sh.L20 * (N.y*N.z) + sh.L21 * (N.y*N.x) + sh.L22 * (N.z*N.x) + sh.L2m2 * (N.x*N.y) + sh.L2m1 * (N.z*N.z - 1.0/3.0); }

SH irradiance 非常便宜（27 floats，幾次乘法），適合靜態場景。

Reflection Probe

多個位置放置 reflection probe，每個 probe 有自己的 environment map + SH。物體採樣最近的 probe。

每個 probe： - position (world space) - irradiance SH - prefiltered environment cubemap - blend radius (與其他 probe 的過渡半徑) 物體渲染時： 1. 找最近的 N 個 probes 2. 根據 blend radius 混合 3. 對每個 probe 計算 IBL 4. 混合結果

建議實施順序

1
Phase 1: 固定天空 ambient color + AO factor。不需要 extra texture。
2
Phase 2: 加入 SH irradiance。烘焙到文件或 runtime 計算。
3
Phase 3: 加入 IBL reflection（prefiltered env cubemap, mipmapped）。
4
Phase 4: Reflection Probe 系統。多個 probe 混合。

Component 設計
