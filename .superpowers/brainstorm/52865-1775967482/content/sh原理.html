Spherical Harmonics（球諧函數）探針原理

用 9 個係數代替 64,800 個像素的環境光照

核心思想

問題

環境光照 L(θ, φ) 是定義在球面上的函數，每個方向都有一個顏色。

儲存方式：64,800 個像素（360° × 180°）

問題：儲存量大，且無法快速插值

SH 的解決方案

把複雜函數分解成**簡單基函數**的加权和：

L(θ,φ) ≈ Σ c₀·Y₀ + c₁·Y₁ + c₂·Y₂ + ... + c₈·Y₈ ↑ ↑ 只有 9 個係數 9 個基函數（SH basis）

儲存：9 × 3 channels = 27 floats ✅

三個頻率帶（L = 0, 1, 2）

每個頻率帶有 (2L+1) 個係數：

頻率帶係數數量物理意義圖示
L=0（SH₀）1 個平均亮度（DC）整體亮度均勻分佈
L=1（SH₁）3 個方向性：上下/左右/前後一個啞鈴形狀
L=2（SH₂）5 個細節：邊緣亮度變化複雜形狀
Band L=0 (1 coefficient): Y₀₀ = 0.282095 ← 圖：均勻的球 Band L=1 (3 coefficients): Y₁₋₁ = 0.488603·y ← 圖：啞鈴（上下） Y₁₀ = 0.488603·z ← 圖：啞鈴（前後） Y₁₁ = 0.488603·x ← 圖：啞鈴（左右） Band L=2 (5 coefficients): Y₂₋₂...Y₂₂ ← 圖：更複雜的形狀

如何計算係數（預處理）

在 Editor 或 runtime 把 environment map（或整個場景）投影到 SH 基函數： for each pixel (θ, φ) in environment map: for L = 0 to 2: for M = -L to L: c[L,M] += L(θ,φ) · Yₗₘ(θ,φ) · sin(θ) 全部遍歷後，27 個係數就計算完畢。

這個過程叫做 SH Projection，是一次性的預計算。

如何使用（遊戲中每幀）

給定法線 N = (nx, ny, nz)，採樣環境光照： float3 color = 0.0; // L=0: 1 個係數 color += sh.L00 * 0.282095; // L=1: 3 個係數 color += sh.L1m1 * ( 0.488603 * ny); color += sh.L10 * ( 0.488603 * nz); color += sh.L11 * ( 0.488603 * nx); // L=2: 5 個係數 color += sh.L2m2 * ( 1.092548 * nx*ny); color += sh.L2m1 * ( 1.092548 * ny*nz); color += sh.L20 * ( 0.315392 * (3*nz*nz - 1)); color += sh.L21 * ( 1.092548 * nx*nz); color += sh.L22 * ( 0.546274 * (nx*nx - ny*ny)); // 結果：N 方向的環境光照顏色 return color;

簡單類比：聲音

複雜聲音

一段 10 秒的音頻 = 441,000 個採樣點（44.1kHz × 10s）

無法壓縮、無法理解、無法插值

SH 比擬

傅立葉變換把音頻分解成不同頻率的正弦波：

儲存：100 個頻率的強度（ amplitude）

好處：可以快速插值、壓縮、理解聲音特徵

SH 就是"光照的傅立葉變換"！

在 Engine 中的應用

烘焙（Editor）

每個 probe 位置： 1. 放置一個假設 camera，捕獲周圍環境的 cubemap 2. 對 cubemap 進行 SH projection，得到 9 個係數 3. 儲存到 comp_environment_probe.sh_irradiance

運行時（遊戲中）

每個 fragment 渲染時： 1. 根據 world position 找最近的 N 個 probes 2. 混合 probes 的 SH 係數 3. 用 SampleSH(N) 得到該方向的環境光照 4. 用於 PBR lighting

為什麼只有 9 個係數就足夠？

細節層次需要係數視覺效果
L=0 僅1 × 3只有整體亮度，無方向性
L=0+14 × 3有上下方向區別（天空亮、地面暗）
L=0+1+2 ✅9 × 3足夠表達複雜光照，包括邊緣變化
L=3+16+ × 3幾乎無視覺改善，儲存卻翻倍

L=2 是"性價比最高"的選擇：足夠好，儲存又小。

Probe 放置策略

最簡單：均勻 Grid 每 10m × 10m × 10m 放置一個 probe 更好的：重要性放置 - 角落、光源附近放置更密集 - 空曠區域放置更稀疏 室外場景： - 只需要 1 個全域 probe（天空光照是均匀的） - 室內每個房間 1 個 probe
